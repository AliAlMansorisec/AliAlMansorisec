```yaml
Name        : Ali Al-Mansori
Handle      : AliAlMansoriSec
Role        : Application Security Engineer
Focus       : Web Applications • APIs • Mobile Apps
Methodology : OWASP WSTG · OWASP API Security Top 10 · OWASP MASTG
Experience  : Since 2023
Platform    : الفكر السيبراني — Arabic Cybersecurity Education
```

I find vulnerabilities before attackers do. My work is built on OWASP standards rather than guesswork — every engagement follows a documented methodology, every finding ships with a proof of concept, a severity rating, and a remediation path a dev team can actually act on.

<table>
<tr>
<td width="33%" valign="top">

**🌐 Web Application Testing**

</td>
<td width="33%" valign="top">

**🔌 API Security Testing**

BOLA, BFLA, mass assignment, excessive data exposure, and the rest of the OWASP API Security Top 10.

</td>
<td width="33%" valign="top">

**📱 Mobile App Testing**

Static & dynamic analysis, insecure storage, SSL pinning bypass, and OWASP MASTG-based methodology.

</td>
